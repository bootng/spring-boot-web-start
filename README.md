
# Spring Boot + Thymeleaf Web App

Article link : https://www.bootng.com/2020/05/spring-boot-web-app-with-thymeleaf.html

## 1. How to build
```
$ mvn install

```


## 2. How to start
```
$ mvn spring-boot:run

```

## 2. Access the Application
```
http://localhost:8080/login
http://localhost:8080/blog

```
